# Project Overview

## Summary
Rookies is a SaaS platform designed as a "Virtual COO" for small and medium-sized Indian businesses. It targets niche markets such as home bakers, kirana stores, and Instagram-first brands. By offering an accessible, un-intimidating dashboard, Rookies aids business owners in managing orders, payments, inventory, and customers seamlessly.

## Key Proposition
A defining characteristic of Rookies is its **WhatsApp-first** approach. Using automated communication workflows managed through platforms like n8n, it captures leads and order data directly from WhatsApp chats and funnels them into actionable system records. This streamlines operations that traditionally required manual data entry, reducing human error and latency.

## Target Audience
- Home Bakers
- Kirana Stores
- Instagram-first Brands
- Local SMEs and Microbusinesses 

## Core Capabilities
- Order and fulfillment tracking
- Inventory parsing and low-stock alerts
- Centralized customer data points
- Payments management
- WhatsApp interaction integration via Webhooks
